# AdaptAIR EasyBuild


EasyBuild related files for software used within the AdaptAIR project. 

Includes EasyConfigs derived from / based on the upstream [easybuilders/easybuild-easyconfigs](https://github.com/easybuilders/easybuild-easyconfigs) repository, where the locally available EB installation (or upstream easyconfigs) does not include the required software versions
